# ican plus
一個知識共向平台，使用者透過提供**專業知識**來獲得獎勵。
![介面](/Demo/logo.jpg)
### 一、目錄結構
* icanPlus
    * static	-- web.py中加載css和js等靜態文件的地方
	* view 
	* Demo -- 成品文件
	  app.py	 主程式
	  config.conf -- 配置檔
	  log.json -- log紀錄
	  loop.py -- 背景程式，系統自動執行
	  vault.py -- vault api function

### 二、環境配置
* python 3.5.1
* Maria 10
